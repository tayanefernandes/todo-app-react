# To Do App
App to learn React and Redux

## How to run Backend

### Requirements
- MongoDB
- Node

```
npm install
```
### Dev mode
```
npm run dev
```
### Production mode
```
npm run production
```

## How to run Frontend

### Requirements

```
npm install
```
### Dev mode
```
npm run dev
```
### Production mode
```
npm run production
```
